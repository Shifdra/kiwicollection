# Kiwi Collection - Trevor Smart

## Configuration
- MySQL v8.0.2
- PHP v7.3
- Composer v2.0.11
- Symfony CLI v4.25.4
- Yarn v1.22.1


## Project setup

###Composer
```
composer install
```

###Yarn
```
yarn install
```

### Compiles front-end for development
```
yarn encore dev --watch
```

### Run the local server
```
symfony server:start
```
